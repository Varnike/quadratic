solves linear equation.

Solves linear equasion and writes solutions into solution strucutre. If linear equasion has no solutions, sets sol state to NO_SOL. If linear equasion has infinity solutions, sets sol state to INF_SOL. If linear equasion has one solution, sets sol state to ONE_SOL, calculate root and sets sol x1 value to root value.

Parameters
cpointer to coefficients structure.
solpointer to solution structure.
